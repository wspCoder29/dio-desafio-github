.NET FRAMEWORK
- O que é?
Ambiente de des e execução focado em windows
-Common language runtime + .NET Class library

-Compilador traduz o cod de alto nivel para uma linguagem intermediária
- CLR - common language runtime, dentro dele tem o JIT - "just in time"? 
O JIT pega o cód intermediário EM TEMPO DE EXECUÇÃO e converte essa linguagem
intermediária em linguagem de máquina.


Características Principais
>Gerenciamento de memória
>Sistema de tipos comum > tipos definidos pelo framework
>Variadas bibliotecas de classes utilitarias 
>Interoperabilidade entre linguagens
>Compatibilidade de versões
>Uso do .NET Standard
